# Beginning Python

Python is a good language to write in because of its simplicity, which makes programming easier than usual as a result, as well as being very fun to write in. It is commonly used for scripting and web programming but as it is a general-purpose language, there is not much of a limit as to what can be written in Python.

Python is a language that follows a trending practice in software construction - it is a reasonably simple language, and its power is utilized best with both first-party and third-party modules, of which there are many.
Python is a dynamic language, performing many actions at run time rather than compilation time -- in fact, Python doesn't have compilation time; it is an interpreted language.

Before the language is looked at from a close-up perspective, a few last things:

  • Python is open source under the PSF license (GPL compatible)
  • It has an interactive interpreter, which is very useful for testing features of the language, and your own functions.
  • Python has well-designed object-oriented design features.

In terms of basic, 'core' features of a language, Python has all the common data types that would be expected in a general-purpose language - numbers, strings, lists, and hash tables (in Python, hash tables are more commonly referred to as 'dictionaries'). As previously mentioned, OOP is supported: Python has classes and multi-inheritance. Errors and exceptions are built in to the language, and mismatched types will raise an exception (e.g. "hello" + 9 is an example of string + int).
A very helpful, whilst not unique, feature of Python is its memory management, removing the need to allocate and free memory by hand in your programs.

# Python's Syntax

Here is an example of Python's simple and clear syntax: The hello, world program:

        print "hello, world!"

Comparing Python to Java - another powerful language that is also beginner-friendly - the language's simplicity can be seen very easily. Because everything in Java has to be in a class, there is boilerplate code that is necessary as well. Here is our hello world program in Java:

        public class HelloWorld {
            public static void main(String[] args) {
                System.out.println("hello, world!");
            }
        }

Unlike many C-Like languages which use {braces} and sem;colons, Python has significant whitespace. This means that the scope of a block of code (like an if statement, a function, or a class) is defined by its indentation, rather than its being wrapped in braces - like the Java code above. If we wanted to create a class called HelloWorld, with the same function main in Python, it would look like this: 

        class HelloWorld:
            def main(args):
                print "hello, world!"
It is the colons that begin the scope of the class or function, and the indentation of code that shows the interpeter which scope of code a line is in - so, the line 'print "hello, world" is in the function main()'s scope, which is in the class HelloWorld's scope.

# Learn Python

If you're interested in learning the language, here are some websites and books I can recommend. None of these sources need programming experience, but the books are still good if you are already a programmer:

  • Codecademy - Good for picking up core syntax
  • Python Programming for the Absolute Beginner - An easily readable book, introduces Python through game development.
  • Learn Python the Hard Way - Very popular, an original approach to learning programming. If I didn't know Python, I'd use this.
