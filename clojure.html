Clojure Programming

Clojure is a dynamic, general-purpose, functional programming language that is a dialect of Lisp, and runs on the JVM. It was created by Rich Hickey and is open source.

Homoiconicity back to top

All Clojure code, just like all lisp code, is built up of actual Clojure data structures. Take a function call, for example:

  (Math/pow 2 8)
  ;; => 256.0

This is a list of different elements - the identifier for the function, Math/pow, and the two arguments 2 and 8. The Clojure REPL or compiler will automatically evaluate this list, and call the Math/pow function on 2 and 8, but if the list is quoted - and therefore not evaluated, the REPL will simply return the actual list itself:

  (quote (Math/pow 2 8))
  ;; => (Math/pow 2 8))

After that, we can evaluate the quoted list, creating a more verbose (and pointless) way of writing Clojure code:

  (eval (quote (Math/pow 2 8)))
  ;; => 256.0

Basic Literals back to top

As Clojure runs on the JVM, all of the basic data types correspond to data types in Java.

  • Clojure strings, e.g. "clojure" are Java strings, e.g. "java".
  • true and false have the same boolean values in Clojure and Java.
  • The 'nothing' value, nil, corresponds to the null value in Java.
  • Characters in Java, for example 'b', are delimited with a backslash in Clojure, e.g. \b

Clojure has a function for finding the class from which something derives, and it is used like this:

  (class 5)
  ;; => java.lang.Long

  (class 3.14)
  ;; => java.lang.Double

  (class "fido")
  ;; => java.lang.String

  (class false)
  ;; => java.lang.Boolean

All the types mentioned above are scalar types- the kind of types that aren't collections. The opposite set of types is called, obviously, collection types

Collection Types back to top

The most common collection-based data structures in Clojure are the following four:

  • Lists, as previously seen: 
        (class (quote ("hello" 44.5 true "goodybe")))
    ;; => clojure.lang.PersistentList
  • Vectors: 
        (class [x y 2 5 "fido"])
    ;; => clojure.lang.PersistentVector
  • Maps: 
        (class {:name "Red" :hex "#FF0000"})
    ;; => clojure.lang.PersistentArrayMap
  • and Sets (mathematical sets): 
        (class #{1 2 3 4 5})
    ;; => clojure.lang.PersistentHashSet
