Lishp

Lishp (etym: English + lisp (a programming language)) is a simple programming language that compiles down to Python, and is designed to be ultra-readable, via close similarity to the syntax of the English language.

A snippet of Lishp code can demonstrate this quality:

  say hello, world!
  set my_name to "Lishp"
  say I am called {my_name}
  set age to 10
  add 1 to age
  say Next year I will be {age}

The say command takes plain, un-quoted text as an argument, and prints the text to standard output. When the value of a variable needs to be included in the text, its identifier is surrounded in braces, as seen in the example above with {my_name} and {age}.

Typing

Lishp is dynamically typed, with variable declaration following the syntax

  set [VAR_NAME] to [VAR_EXPR]

where [VAR_EXPR] is an expression, usually just a literal number or string, however any mathematical expression could be used, such as

4 × 5 − 8 + ({x} × 3)

The evaluation is handled by Python's eval() function, but variable identifiers are swapped out for their values beforehand: variables can be used in a mathematical expression with the usual braces syntax.

Strings

When a variable is assigned to a string value, the value is surrounded in double quotes. Concatenation of strings does not require a special character such as a + or , symbol, but is done as such:

  set home to "United" " " "Kingdom"

Concatenation of string identifiers can also be done in the same way:

  set name to "Josh"
  set surname to "Hartigan"
  set full_name to {name} " " {surname}

  // the following will print "Josh Hartigan".
  say full_name
