# Find the Most Frequent Integer in an Array

Mathematically, this problem would be rephrased as 'Find the Mode of an Array'. So, the function we write to solve the problem will be named mode().

The function will take an array of integers as its argument, and simply return the element that appears the most in the array. So, how do we build an algorithm to do that?

An efficient, and considerably easy, method would be to use a hash table. If the mode() function is given an array of 2 integers x and y, it will create a table like this:

Where x and y are the elements of the array, and a and b are the number of times they occur in the array respectively. To calculate the frequencies of an array's elements, like a and b, we must iterate through the elements and follow this routine:

  • Check if the array's element is already in the hash table. We will call this boolean in.
  • If in is true - and the element's already in thetable - simply iterate the element's value in the table - that is, element:occurrences becomes element:occurences + 1.
  • Otherwise, if in is false, and the element isn't in the table, place it in the table with a value of 1, representing 1 occurrence.

The image below shows how the routine applies to the list [3, 6, 4, 1, 9, 8, 3, 2, 1, 3], at each point that the loop's iteration is at a 3:

At first, in is false, because 3 is not yet in the hash table. So, the algorithm puts it in the table with a value of 1 (represented in the image by {3:1}.

At the next iteration at a 3, in is true, of course. So rather than placing the element in the table, the algorithm increments its value - {3:1} becomes {3:2}. The same thing happens at the last appearence of 3, incrementing its value by 1 again, to become 3.

After all of that algorithmicity, we are left with a hash table of array elements and their frequencies. If we used the array shown in the last image, our table would be {3:3, 6:1, 4:1, 1:2, 9:1, 8:1, 2:1}.

What we need our mode() function to calculate now is which element (or key) has the highest value. Here's the routine to follow for that:

  • Define a variable with a name like maxKey, initial value of 0.
  • Iterate through each key of the hash table (i.e. 3, 6, 4, 1, etc.)
  • For each key in the table, if it's value is bigger than maxKey, maxKey's value get's changed to the key (not the key's value).

After the loop described above, `maxKey` will store the element of the array that has the highest frequency (in our example's case, 3). The `mode()` function will return it, and it's finished!
